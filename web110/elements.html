HTML Elements

Element Description Block, Inline, N/A Container or Empty
<!--  --> HTML comments are invisible on a browser unless you view its source. They're used to provide instructions and credit. N/A Container
<!DOCTYPE> An instruction to the web browser about what version of HTML the page is written in. N/A Empty
<a> Defines a hyperlink, which is used to link from one page to another. Inline Container
<article> Specifies independent, self-contained content. Block Container
<aside> Defines some content aside from the content it's placed in. Block Container
<body> Defines the document's body Block Container
<br> Inserts a single line break. Inline Empty
<dd> Used to describe a term/name in a description list Inline Container
<div> Defines a division or a section in an HTML document. Block Container
<dl> Defines a description list. Block Container
<dt> Defines a term/name in a description list. Inline Container
<em> Renders and emphasized text. Inline Container
<fieldset> Used to group related elements in a form. Block Container
<figcaption> Defines a caption for a <figure> element. Inline Container
<figure> Specifies self-contained content like illustrations, diagrams, photos, code listings, etc. Block Container
<footer> Defines a footer for a document or section. Block Container
<form> Used to create an HTML form for user input. Block Container
<h1> to <h6> Used to define HTML headings. Inline Container
<head> A container for all the head elements. Block Container
<header> Represents a container for introductory content or a set of navigational links. Block Container
<html> Tells the browser that this is an HTML document. Block Container
<iframe> Specifies an inline frame. Inline Container
<img> Defines an image in an HTML page. N/A Empty
<input> Specifies an input field where the user can enter data. N/A Empty
<label> Defines a label for an element that provides usability improvement for mouse users. Inline Container
<li> Defines a list item. Inline Container
<link> Defines a link between a document and an external resource. N/A Empty
<meta> Provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable. N/A Empty
<nav> Defines a set of navigation links. Block Container
<ol> Defines a numerical or alphabetical ordered list. Block Container
<p> Defines a paragraph. Block Container
<script> Used to define a client-side script. It either contains scripting statements, or it points to an external script file. Block Container
<section> Defines sections in a document, such as chapters, headers, footers, or any other sections of the document. Block Container
<strong> A phrase tag that defines important text. Inline Container
<style> Used to define style information for an HTML document. Block Container
<table> Defines an HTML table. Block Container
<td> Defines a standard cell in an HTML table. Inline Container
<th> defines a header cell in an HTML table. Inline Container
<title> Required in all HTML documents to define the title of the document. Inline Container
<tr> Defines a row in an HTML table. Block Container
<ul> Defines an unorderd list. Block Container
